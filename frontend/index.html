Disk Organiser

Visualise and safely organise files on your hard drive — previews, backups, and optional AI suggestions.

Repository Download Usage Docs

About

Disk Organiser is a small prototype to help users visualise, search and safely reorganise files on their filesystem. Operations create automatic backups so changes can be reviewed or undone.

Screenshots

Quick start

  1. Download the latest release from the releases page or run from source.
  2. Start the backend: python backend/app.py
  3. Open this file in a browser: frontend/index.html (the UI expects the API at http://127.0.0.1:5000 by default).

# User-Guided Organisation Agent Customization

## Purpose
Helps users create a custom organisation structure for their hard drive, tailored to their needs and preferences.

## Usage
- Example prompt: "Help me organise my downloads folder for work and personal files."
- The agent interacts with the user to gather requirements, suggests folder structures, and can automate file moves (with confirmation).

## Implementation Notes
- Backend logic for file/folder operations and user preference storage.
- Web UI guides the user step-by-step, with clear, minimal prompts and undo options.
- Supports both local and remote model recommendations for organisation strategies.

## Extensibility
- Add templates for common organisation schemes (e.g., by project, date, type).
- Integrate with duplicate search and visualisation agents for holistic organisation.
